# chip8 emulator

*work in progress*

To run the emulator you'll need to install [SDL2](https://www.libsdl.org/download-2.0.php).


## Usage
```
$ git clone git@github.com:fmmartins/chip8.git
$ cargo install && cargo run -- <file>
```
